# Image assets

Drop screenshots and other marketing / docs images here.

Suggested filenames (used by the root `README.md`):

| File | Suggested content |
|------|-------------------|
| `main-window.png` | Full app window with source, target, and tracks loaded |
| `destination.png` | Close-up of the Destination / save location section |
| `mux-complete.png` | Successful mux (log panel / result) |
| `duration-mismatch.png` | Duration mismatch confirm dialog (source vs target lengths differ) |

Use PNG or WebP. Keep widths around 1200–1600px so they look sharp on GitHub.
